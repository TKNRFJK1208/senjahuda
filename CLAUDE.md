# 千社札ポートフォリオサイト

## プロジェクト概要
佐藤舞優氏の千社札・茶会印刷物のポートフォリオサイト

## 技術スタック
- Nuxt 3（SSG）
- microCMS（作品ギャラリー・Q&A）
- Netlify（ホスティング）
- TailwindCSS

## コーディング規約
- 出力はすべて日本語で行う
- コメントは日本語で記述
- 変数名・関数名は英語（camelCase）
- インデント：スペース2つ
- コンポーネントは /components 配下に配置

## デザイントークン
- メインカラー：江戸朱 #8B0000
- サブカラー：墨 #1A1A1A
- 背景：生成り #FAF6EF

## 注意事項
- SEO最優先。useServerSeoMetaを必ず使う
- 画像はすべて @nuxt/image 経由で最適化
- microCMS APIキーは .env から参照すること
```

CLAUDE.mdに「日本語で出力する」と記述するだけで常に日本語で応答してくれます。 これを書いておくだけで、毎回説明する手間がゼロになります。
```

```
## STEP 5：Plan Mode の使い方（大きなタスク前に必ず使う）

大規模な開発タスクでは、いきなりコーディングに入るより事前設計が重要です。Shift+TABキーを2回押すとPlan modeに切り替わり、実装方針を明確にしてから作業を進められます。 
```
# 例：フェーズ2の作品ギャラリー実装前
「作品ギャラリーページを実装したい。
まず実装計画だけ出して、コードはまだ書かないで。」
```

AIが計画を提示してくれるので、**OKなら実装へ進む**、**問題があればその場で修正**できます。承認してからコードを書かせる流れが安全です。
```
